## Projeto: API de Álbuns com FastAPI

### Features:

- API REST assíncrona
- Validações com Pydantic
- Rotas com tratamento de erros
- Estrutura modular
- Simulação de banco com lista

### Exemplo de chamada POST:

```json
{
  "id": 1,
  "title": "Origin of Symmetry",
  "artist": "Muse",
  "year": 2001,
  "genre": "Alternative Rock"
}
```
