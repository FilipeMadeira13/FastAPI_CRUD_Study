# Projeto FastAPI: Music Album CRUD

## Endpoints

- `GET /albums`: lista todos
- `POST /albums`: adiciona novo
- `GET /albums/{id}`: busca por ID
- `PUT /albums/{id}`: edita
- `DELETE /albums/{id}`: remove

## Funcionalidades

- Código assíncrono
- Validação com Pydantic
- Erros tratados com HTTPException
